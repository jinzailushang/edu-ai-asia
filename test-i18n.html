测试步骤说明

步骤 1: 浏览器语言检测

  1. 清除所有 localStorage 和 sessionStorage 数据
  2. 访问网站首页
  3. 检查是否根据浏览器语言自动跳转到对应语言版本

步骤 2: 语言偏好保存

  1. 在网站上切换语言
  2. 检查 localStorage 中是否保存了语言偏好
  3. 刷新页面，验证语言是否保持

步骤 3: 语言切换功能

  1. 点击语言切换按钮
  2. 验证 URL 是否正确更新
  3. 验证页面内容是否切换到目标语言
  4. 验证语言偏好是否保存

实现代码示例

// 语言检测函数 function detectBrowserLanguage() { const browserLang = navigator.language.toLowerCase(); if (browserLang.startsWith('zh')) return 'zh'; if (browserLang.startsWith('en')) return 'en'; return 'zh'; // 默认中文 } // 保存语言偏好 function saveLanguagePreference(lang) { localStorage.setItem('preferred-language', lang); } // 获取语言偏好 function getLanguagePreference() { return localStorage.getItem('preferred-language') || detectBrowserLanguage(); }
